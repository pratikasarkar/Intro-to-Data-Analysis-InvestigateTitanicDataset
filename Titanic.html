The above data shows us that there are many missing values for the fields like age,cabin and embarked. Also after looking into the dataset I found that many values for the Fare field are set to 0.¶

To fill in the missing values for age field, the way I find best is to group the data by their salutations which can be extracted from the name field and set the median of the grouped data to the missing values. Taking median seems the best way so that it neglects the effect of any outliers in the data. This might not be the best way of predicting the age, but the data does not provide with any additional information.¶

Filling the missing values for cabin and embarked fields does not seem necessary as for as the scope of this particular investigation is taken into consideration.¶

To fill in the missing values for fare field, the best way is to group data by passenger class and set the median of the grouped data to the missing class. Again, the median used is for the purpose of overlooking the outliers.¶

The below function replace_fare checks for the entries having fare as 0 or empty and replaces it with median of fare belonging to a particular class¶

The median has been used so that there are no effects of outliers on the dataset¶

In [5]:
def replace_fare(grp):
    grp['FareNew'] = np.where(((grp['Fare'] ==0) | (grp['Fare'].isnull())), grp['Fare'].median(),grp['Fare'])
    return grp

def name_extract(word):
    return word.split(',')[1].split('.')[0].strip()

titanic_df = pd.merge(titanic_df,df,left_index = True, right_index = True)

The above function name_extract takes record's name field as input and splits it by comma to get the 2nd part of he name field which appears after comma and again splits it using dot to get the 1st part before the dot. Using strip we remove all the spaces which could be present before or after the salutation.¶

Using pd.merge we add the salutations as an additional field to the original dataset¶

Report¶

Investigation for the questions posed¶

To find the tentative fare for each class, finding average seemed to be the best option.¶

Even for finding the age for each class, the average has been calculated.¶

To find the number of family members, for each passenger, it has been calculated by 1D vector addition of two columns representing spouse-siblings and parent-children.¶

Then the data is grouped depending on the number of family members.¶

For the proportion of male and female casualties, the dataset was first grouped by column 'Survived' and furthur grouped by column 'Sex'.¶

The proportion was then calculated by number of male or female casualties compared to total number of male or female respectively.¶

The male and female casualties were futhur grouped by passengers class to find out what proportion of casualties belonged to what class.¶

Necessary data wrangling¶

While investigation the dataset, there were some problems which highlighted and were supposed to be fixed to find the accurate analysis for the dataset¶

Some entries for the column 'Fare' seemed to have value 0 or had a null value. To fix this issue, median of fare depending on the passenger class was calculated and was set as the value for all those columns which were blank or had value as 0.¶

Similarly, some entries for the column 'Age' seemed to have a null value. To fix this issue, median of age depending on the salutation of the passenger from its name was calculated and was set as the value for all those columns which were blank.¶

To find the number of family members onboard for each passenger, 1D vector addition of columns 'SibSp' and 'Parch' was done.¶

Limitations of analysis¶

The analysis does not clearly head into finding the cause of titanic incident and its casualties.¶

The is no proper way deduced during the analysis to do statistical testing on it. Due to this, all the conclusions are made purely based on how the data looks on the plots and histograms, which can be misleading many times.¶

The analysis only shows the number of family members onboard for the passengers who survived and those who did not. There is no way to tell from the analysis how many family members actually survived and what were the number of casualties of family members of a particular passenger.¶

The data being analysed is just a random sample of the actual data. So this means there could be other samples of such data too which could shift the paradigm of the population completely other way. Thus we cannot imply causation depending on this particular dataset.¶

Summary¶

The plot between passenger class and average fare shows the the fare variation depending on the class of passenger. We can see from the plot that the average fare for 1st class passenger is much higher than class2 and class3 passengers¶

The histograms for number of passengers survived and number of casualties depending on the number of family members onboard helps us analyze whether the family was big family of a small family. The plot shows us that there were more casualties for passengers with less or no family members onboard.¶

The proportion of male and female casualties shows us that if an incident like titanic was to happen again, then there could be a chance more number of male casulties than female casulties.¶

The casualties depending on passenger class also shows us that the security arrangements for 1st class paseengers might have been better as compared 2nd class and 3rd clasd passengers.¶
